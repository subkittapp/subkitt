# SubKitt

You ship. SubKitt turns your work into inbound.

Grow a founder audience without becoming a content person. For technical founders who'd rather stay in the code.

## Status

Day 44 of building in public. v1 is live. Currently onboarding early builders and implementing dynamic admin controls.

## Tech

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Supabase Database & Native GitHub OAuth
- Google Gemini, OpenAI GPT, and Anthropic Claude LLM integrations
- Resend Email automation pipeline
- Embedded interactive Administration Panel Console
